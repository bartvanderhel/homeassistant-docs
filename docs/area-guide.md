# Home Guide Per Room

This page is written as a simple home guide. It starts with each area (room), then lists all devices in that area, and what happens automatically for each device.

How to read this page:

- **Automatic actions**: things Home Assistant does by itself (automations).
- **Button actions / scenes**: actions started by scripts (for example mood lights or all lights off).
- **No linked automation or script yet**: the device is visible in Home Assistant but not directly used in your YAML automations/scripts.
