# Skills & Automation Support

This repository includes a collection of agent "skills" used to assist with developing, reviewing, and maintaining Home Assistant configuration.

Location: `.agents/skills/`

Installed examples:

- `home-assistant-best-practices` — guidance for safe Home Assistant configuration and refactoring.
- Superpowers collection — `using-superpowers`, `systematic-debugging`, `test-driven-development`, `writing-plans`, and more.

How to use:

- Read a skill's `SKILL.md` to learn its workflows and prompts.
- Use the skills locally to plan changes, run verification steps, and request reviews before merging.
