# Home Assistant — Documentation

This documentation site describes the Home Assistant instance stored in this repository.

Key locations in the repository:

- `configuration.yaml` and the `configuration/` folder — main Home Assistant configuration.
- `.agents/skills/` — local skills and guidance used for maintenance and automation.
- Git hooks / git-ha-ppens integration — repository is synchronized with the running Home Assistant instance in your environment.

This site is generated with MkDocs and includes basic overviews of configuration and installed skills. Use the Docker image to serve locally (see the project root `Dockerfile`).
